# Video Editor Portfolio Website – Azeem Naveed

## Project Overview

This project is a modern portfolio website for **Azeem Naveed**, a video editor based in **Kandy, Sri Lanka**.

The purpose of this website is to present Azeem’s video editing work, services, past projects, client reviews, freelance platform links, and contact details in a professional and visually attractive way.

# 8. Submit a Review Section

The website should include a place where users or clients can submit reviews.

## Review Form Fields

The review form should include:

* Name
* Email
* Project Type
* Star Rating
* Comment / Review
* Submit Review button

## Star Rating Feature

The star rating should be interactive.

Users should be able to click:

* 1 star
* 2 stars
* 3 stars
* 4 stars
* 5 stars

The selected rating should be visually highlighted.

## Review Submission Message

After submitting a review, show a message such as:

> Thank you for your review. Your feedback has been submitted and will appear after approval.

## Review Moderation Requirement

Reviews should not appear publicly immediately unless approved.

Recommended approach:

* Store submitted reviews first
* Mark them as `pending`
* Admin can approve or reject reviews
* Only approved reviews should display on the website

This helps prevent spam, fake comments, or inappropriate content from appearing on the portfolio.

## Review Data Structure Example

```json
{
  "name": "Client Name",
  "email": "Client Email",
  "projectType": "Project Type",
  "rating": 5,
  "comment": "Client review comment",
  "status": "pending",
  "createdAt": "Date"
}
```

Only reviews with the following status should be shown publicly:

```json
{
  "status": "approved"
}
```
